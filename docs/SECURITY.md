# Política de Segurança

## Reportando Vulnerabilidades
A segurança do NutixoApp é nossa prioridade. Se você descobrir uma vulnerabilidade de segurança, por favor, envie um reporte detalhado para a equipe técnica.

**Não abra issues públicas para problemas de segurança.**

### Processo de Reporte
1. Envie um e-mail para `security@nutrixo.com` (Exemplo).
2. Forneça detalhes sobre a falha e passos para reprodução.
3. Nossa equipe responderá em até 48 horas.

## Versões Suportadas
| Versão | Suporte de Segurança |
| :--- | :--- |
| 1.1.x | ✅ Ativo |
| 1.0.x | ❌ Apenas patches críticos |
