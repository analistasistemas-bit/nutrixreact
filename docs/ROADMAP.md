# Roadmap Estratégico: Nutixo Elite 🗺️

Este roadmap define as etapas de evolução do NutixoApp do estado atual para um produto líder de mercado ("Elite"), baseado no documento de Auditoria de Produto.

---

## 🟢 Etapa 1: Fundação PWA & Portabilidade (Q1 2026)
*Foco: Transformar o site em um App instalável e resiliente.*
- [ ] **PWA Core**: Implementação do `vite-plugin-pwa` com manifesto e ícones adaptativos.
- [ ] **Offline Ready**: Configuração de Service Workers para cache de App Shell.
- [ ] **Ajustes de Safe Area**: Otimização de layouts para telas de iPhones (Notch) e gestos Android.
- [ ] **Custom Splash Screen**: Interface de carregamento profissional com a marca Sentinel.

## 🟡 Etapa 2: Infraestrutura Full Stack Real (Q2 2026)
*Foco: Sair da simulação (mocks) para dados reais e protegidos.*
- [ ] **Migração de Dados**: Integração total com o banco de dados Insforge DB para XP e Exames.
- [ ] **Segurança Avançada**: Implementação de Autenticação Real com MFA (Multi-fator).
- [ ] **Analytics Pro**: Integração com ferramentas de monitoramento de jornada do usuário.
- [ ] **Fila de Notificações**: Configuração de canal para Push Notifications (mesmo com app fechado).

## 🟠 Etapa 3: Refinamento UX Pro Max & Acessibilidade (Q3 2026)
*Foco: Excelência visual e inclusão digital.*
- [ ] **Tipografia Técnica**: Atualização global para `Fira Sans` e `Fira Code`.
- [ ] **Feedback Ativo**: Implementação de Skeleton Screens em todas as áreas de carregamento de IA.
- [ ] **Inclusão Digital**: Auditoria e correção de Acessibilidade (WCAG) para suporte a leitores de tela.
- [ ] **Modo OLED**: Interface ultra-dark para economia de bateria e conforto visual extremo.

## 🔴 Etapa 4: Diferenciais Competitivos (2027+)
*Foco: O "Wow Effect" que destaca o produto.*
- [ ] **Sentinel AR**: Visualização do Pet Sentinel em Realidade Aumentada.
- [ ] **Widgets Sentinel**: Resumo de saúde e metas direto na Home do iOS/Android/Windows.
- [ ] **IA Preditiva**: Sugestões proativas de saúde baseadas em tendências históricas.

---
👉 *Este roadmap substitui a versão anterior e reflete a visão de Auditoria de Elite.*
