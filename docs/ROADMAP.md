# Roadmap de Produto - NutixoApp 🗺️

Visão estratégica para a evolução da plataforma NutixoApp.

## 🟢 Curto Prazo (Q1 2026) - Consolidação
- [ ] **Melhoria da extração de IA**: Aumentar a precisão em exames de imagem complexos.
- [ ] **Feedback do Pet**: Implementar mais 3 estados emocionais baseados em metas batidas.
- [ ] **Dashboard Mobile**: Otimizar tabelas de biomarcadores para telas pequenas.

## 🟡 Médio Prazo (Q2-Q3 2026) - Expansão
- [ ] **Integração Apple Health/Google Fit**: Sincronização automática de passos e sono.
- [ ] **Comunidade Nutixo**: Ranking de XP entre amigos e desafios em grupo.
- [ ] **Modo Offline**: Cache de dados para consulta de planos alimentares sem internet.

## 🔴 Longo Prazo (2027+) - Ecossistema
- [ ] **App Nativo (iOS/Android)**: Migração total para React Native.
- [ ] **API para Nutricionistas**: Portal dedicado para profissionais acompanharem seus pacientes.
- [ ] **Machine Learning Preditivo**: Sugestões de saúde baseadas em tendências históricas de meses.

---
*Este roadmap é flexível e evolui com o feedback dos usuários.*
