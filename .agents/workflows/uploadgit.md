---
description: Executa o fluxo de upload padrão do projeto Nutrixo
---

1. Execute o comando `git status` para visualizar as alterações.
2. Adicione todos os arquivos com `git add .`.
3. Faça o commit com a mensagem "Atualizacao" usando `git commit -m "Atualizacao"`.
// turbo
4. Envie para o servidor com `git push`.
